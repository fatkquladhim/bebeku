# Folder Gambar Peternakan

Letakkan foto-foto peternakan Anda di folder ini dengan nama file berikut:

## Nama File yang Diharapkan:

- `farm-1.jpg` - Foto utama kandang bebek / fasilitas
- `farm-2.jpg` - Foto hasil panen telur
- `farm-3.jpg` - Foto DOC (anakan bebek)
- `farm-4.jpg` - Foto proses pemberian pakan
- `farm-5.jpg` - Foto tim peternak
- `farm-6.jpg` - Foto proses pengemasan

## Format yang Didukung:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

## Tips:
- Gunakan gambar dengan resolusi tinggi (minimal 1920x1080)
- Pastikan gambar terlihat profesional dan menarik
- Caption dapat diedit di file `app/(public)/page.tsx` pada bagian `FARM_DATA.gallery`

## Caption Default:
1. "Kandang bebek modern dengan sistem ventilasi otomatis"
2. "Hasil panen telur bebek segar setiap hari"
3. "DOC berkualitas dengan vaksin lengkap"
4. "Pemberian pakan berkualitas untuk kesehatan optimal"
5. "Tim peternak profesional dan berpengalaman"
6. "Proses pengemasan telur dengan standar kebersihan tinggi"

Setelah menambahkan gambar, refresh halaman untuk melihat perubahan.
